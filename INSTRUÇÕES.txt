📋 PASSO 1: PREPARAR O AMBIENTE
───────────────────────────────

Se é a PRIMEIRA VEZ que usa a ferramenta:
  1. Clique duplo em: TESTAR.py
  2. Isso vai verificar se tudo está instalado corretamente
  3. Se algum teste falhar, siga as instruções que aparecem
  4. Se todos passarem, continue para o Passo 2


📝 PASSO 2: CRIAR ARQUIVO COM LINKS
───────────────────────────────────

1. Crie um arquivo novo chamado "meus_videos.txt" (ou use exemplo_links.txt)

2. Abra em Bloco de Notas ou editor de texto

3. Cole seus links, um por linha:

   https://www.tiktok.com/@usuario/video/123456
   https://www.youtube.com/watch?v=ABC123
   https://www.instagram.com/reel/XYZ789/

4. Salve o arquivo

💡 DICA: Você pode clicar com botão direito em um vídeo
        → "Copiar link" → colar no arquivo de texto


🚀 PASSO 3: EXECUTAR A FERRAMENTA
─────────────────────────────────

OPÇÃO A: Duplo clique simples
  → Clique 2 vezes em: EXECUTAR.bat

OPÇÃO B: Rodar manualmente
  → Abra CMD/Terminal na pasta
  → Digite: python video_downloader.py


🎯 PASSO 4: USAR A INTERFACE
───────────────────────────

Na janela que abrir:

1. SELECIONAR ARQUIVO
   → Clique em "Selecionar" (ao lado do arquivo de links)
   → Escolha seu arquivo .txt com os links

2. ESCOLHER PASTA
   → Clique em "Procurar" (ao lado da pasta de destino)
   → Escolha onde salvar os vídeos
   → OU deixe padrão: C:\Users\SEU_USUARIO\Downloads\Videos

3. INICIAR
   → Clique no botão verde "▶ INICIAR DOWNLOADS"
   → Acompanhe o progresso no log abaixo

4. AGUARDAR
   → Não feche a janela enquanto baixa!
   → Leia o log para ver o progresso
   → Pode demorar vários minutos dependendo da quantidade

5. ABRIR PASTA
   → Clique em "🔍 Abrir pasta de destino" para ver os vídeos baixados


⚙️  CONFIGURAÇÕES IMPORTANTES
────────────────────────────

✓ A ferramenta baixa SEMPRE na MELHOR QUALIDADE disponível
✓ Combina melhor vídeo + melhor áudio automaticamente
✓ Funciona com: YouTube, TikTok, Instagram e 500+ outros sites
✓ Nomeia arquivos com título original do vídeo


❌ ERROS COMUNS E SOLUÇÕES
──────────────────────────

ERRO: "yt-dlp não encontrado"
SOLUÇÃO:
  → Abra CMD
  → Digite: pip install yt-dlp -U
  → Aguarde terminar
  → Tente novamente

ERRO: "Interface abriu mas nada baixou"
SOLUÇÃO:
  → Clique em TESTAR.py para diagnosticar
  → Verifique se links estão corretos (sem espaços extras)
  → Verifique permissão de escrita na pasta de destino

ERRO: "Vídeo privado / removido"
SOLUÇÃO:
  → O vídeo não está mais disponível
  → Remova esse link do arquivo
  → Tente com outro vídeo

ERRO: "Pasta não existe"
SOLUÇÃO:
  → Verifique o caminho da pasta
  → Pode estar digitado errado
  → Use o botão "Procurar" para selecionar
